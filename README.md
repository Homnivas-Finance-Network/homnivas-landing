# Homnivas - Homes Reimagined

Homnivas is a premium real estate transformation studio. We breathe life into tired bricks, transforming undervalued spaces and old structures into bespoke, high-end, ready-to-move havens.

## 🚀 Live Demo
https://homnivas.space

## 🛠️ Tech Stack
This project is a high-performance, single-page static website built for maximum speed and easy deployment:
* **HTML5**
* **Tailwind CSS** (via CDN for rapid styling)
* **Google Fonts** (Inter & Poppins)
* **Lucide Icons** (via CDN)
* **Vanilla JavaScript** (for mobile menu and modal interactions)

## 📂 Deployment
This site is designed to be deployed instantly on **Cloudflare Pages** with zero build configuration.
1. Connect this repository to Cloudflare Pages.
2. Select **None** for the build framework.
3. Leave the build command and output directory blank.
4. Deploy! The site will serve the `index.html` file directly.
